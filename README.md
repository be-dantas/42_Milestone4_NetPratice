# 42_Milestone4_NetPratice

## Descrição

O **NetPractice** é um projeto prático da escola 42 cujo objetivo é introduzir os conceitos fundamentais de redes de computadores. O projeto consiste na resolução de exercícios interativos onde é necessário configurar corretamente endereços IP, máscaras de sub-rede e gateways, além de compreender o papel de dispositivos como roteadores e switches para que uma rede funcione corretamente.

A atividade é realizada por meio de uma interface web que simula pequenas redes, permitindo ao estudante testar e validar suas configurações.

## Instruções

### Execução da interface de treinamento

1. Baixe o arquivo fornecido na página do projeto NetPractice.
2. Extraia os arquivos em um diretório de sua preferência.
3. Abra o arquivo `index.html` em um navegador web.
4. Insira seu login da 42 no campo indicado para praticar com sua configuração pessoal.
5. Resolva os níveis ajustando os campos editáveis até que a rede funcione corretamente.
6. Utilize o botão **Check again** para validar a configuração.

### Exportação das configurações

- Após concluir cada nível com sucesso, utilize o botão **Get my config** para exportar o arquivo de configuração correspondente.
- Exporte **um arquivo por nível** antes de avançar para o próximo.

### Submissão

- O projeto possui **10 níveis**, portanto devem ser entregues **10 arquivos de configuração**, sendo **um por nível**.
- Todos os arquivos exportados devem estar localizados **na raiz do repositório Git**.
- Apenas os arquivos presentes no repositório serão considerados durante a avaliação.

## Recursos

### Conceitos de redes estudados

- **Endereçamento TCP/IP**: definição de endereços IP únicos para cada dispositivo na rede, permitindo a identificação e a comunicação entre hosts.
- **Máscara de sub-rede**: utilizada para dividir uma rede em sub-redes menores, determinando quais bits do endereço IP representam a rede e quais representam os hosts.
- **Gateway padrão**: endereço do roteador responsável por encaminhar o tráfego para fora da rede local quando o destino não pertence à mesma sub-rede.
- **Roteadores e switches**: dispositivos de rede responsáveis, respectivamente, pelo roteamento de pacotes entre redes distintas e pela interconexão de dispositivos dentro da mesma rede local.
- **Comunicação entre dispositivos em rede**: princípios básicos de troca de dados entre hosts, respeitando regras de endereçamento e roteamento.
- **Camadas do modelo OSI**: modelo conceitual que divide a comunicação em rede em camadas, facilitando o entendimento do funcionamento e da resolução de problemas em redes de computadores.
